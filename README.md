# Convert variable, Translate text

## Description

Support for quickly converting variable writing style, converting text to variable writing style, translating selected text, ...

## Features

##### Open select MENU

    -   Win: ctrl+alt+F,
    -   mac: shift+cmd+F
    or
    -   Win: ctrl+alt+C,
    -   mac: shift+cmd+C

##### Translate

    -   Win: ctrl+alt+T,
    -   mac: shift+cmd+T

##### HTML To JSX

    -   Win: ctrl+alt+X,
    -   mac: shift+cmd+X

##### Add console.log()

    -   Win: ctrl+alt+L,
    -   mac: cmd+alt+L

## Requirements

Select at least 1 text
